## PostgreSQL

### How to run the test

- Prerequisites:

  - `psycopg3` - https://www.psycopg.org/psycopg3/docs/basic/install.html

- Set your environment variables for:

  - PGHOST
  - PGPORT
  - PGDATABASE
  - PGUSER
  - PGPASSWORD

- Run locust as usual, see https://docs.locust.io/en/stable/quickstart.html
